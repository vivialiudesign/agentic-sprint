
# Agentic Sprint

A structured 5-phase multi-agent design sprint skill for Claude.
Not a faster design sprint — a decision system for AI-native design work.

## Install

**Claude Code (personal)**
```bash
unzip agentic-sprint.skill -d ~/.claude/skills/agentic-sprint
```

**Claude Code (project)**
```bash
unzip agentic-sprint.skill -d .claude/skills/agentic-sprint
```

**Cursor**
```bash
unzip agentic-sprint.skill -d ~/.cursor/skills/agentic-sprint
```

**claude.ai**
Settings → Customize → Skills → upload `agentic-sprint.skill`

## Trigger phrases
- "let's run an agentic sprint"
- "agentic sprint on [problem]"
- "run a sprint"

## Phases
1. Define — problem sensemaking + principles
2. Discover — agent research + affinity mapping
3. Explore — 16 ideas + sketch wireframes + voting
4. Converge — group solution + story test + journey map
5. Build — deck + screens + brief + quality score
